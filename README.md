# Description
This is simple service for writing down notes from client to server - and every time we write we gonna rewrite the old note.

Full explanation about this repo is in here - medium link

# Features
### GET /read
endpoint to read notes

### POST /writes
endpoint to write notes (by doing this, it  will rewrite the old notes)

### GET /health
to  ensure that our server is still alive, just in case hahaha!
